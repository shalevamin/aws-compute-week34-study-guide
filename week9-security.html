בקרת כיסוי

כל הנושאים שנכנסו לעמוד

הרשימה הזאת נועדה לוודא שלא נשארו מושגים, טבלאות, שאלות דוגמה או תרשימי תהליך מהמצגת בלי מענה בתוך האתר.

  • יעדי השיעור: NACL, Security Groups, NAT, DDoS, WAF, KMS
  • חזרה על VPC, Subnets, VPC Peering, Transit Gateway, PrivateLink ו־VPN
  • חומת אש, Rules, Allow, Deny/Reject, מקור, יעד, פורט ופרוטוקול
  • NACL כחומת אש סטטית ברמת Subnet
  • סדר כללים ב־NACL ו־Allow/Deny
  • Default NACL: Inbound ו־Outbound פתוחים, עם Deny סופי
  • Custom NACL: תעבורה חסומה עד להגדרת כללים
  • Stateless packet filtering ודוגמת הבקשה/תגובה
  • Security Group כחומת אש ברמת משאב או Instance
  • Security Group Stateful והחזרת תגובה ללא כלל Inbound חדש
  • SG על EC2, RDS ו־EBS
  • ברירות מחדל של SG ותעבורה בתוך הקבוצה עצמה
  • טבלת NACL מול SG: Inbound, Outbound, Default, Custom, State
  • שאלות בדיקת ידע מהמצגת ופתרונותיהן
  • שאלת מבחן לדוגמה על Default NACL
  • DoS ו־DDoS, בוטים, הצפה ושלילת שירות
  • Volumetric, Protocol ו־Application Layer attacks
  • AWS Shield Standard מול Shield Advanced
  • Shield tiers: ניטור, L3/L4, DRT, cost protection, Layer 7 anomaly
  • AWS WAF כ־Web Application Firewall ברמת L7
  • Web ACL, Rule groups, Priority ו־Managed Rules
  • SQL Injection, XSS, Rate Limiting וחסימה לפי IP
  • WAF על ALB, API Gateway, CloudFront, AppSync ו־Cognito User Pool
  • WAF לא תומך ב־Network Load Balancer; שימוש ב־Global Accelerator ל־IP קבוע
  • השוואת Shield מול WAF
  • NAT Gateway, Private Subnet, Public Subnet, EIP ו־Route Tables
  • תרשים NAT הישן וזרימת NAT Gateway הנוכחית
  • KMS, הצפנה במנוחה והצפנה במעבר
  • יצירה או יבוא מפתחות, BYOK, Key Policy, IAM, CloudTrail
  • HSMs ו־FIPS 140-3 Level 3
  • Envelope Encryption, Data Key מוצפן ולא מוצפן
  • מחזור חיי מפתחות: Create, Store, Encrypt/Decrypt, Integrate, Manage, IAM
