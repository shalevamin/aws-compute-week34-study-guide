בקרת כיסוי

כל הנושאים שנכנסו לעמוד

הרשימה הזאת נועדה לוודא שכל נושא, דוגמה, צילום מסך ותהליך שהופיעו במצגת קיבלו ייצוג טקסטואלי או תרשים HTML בתוך העמוד.

  • רשימת נושאי השיעור: CloudTrail, Inspector, GuardDuty, CloudWatch, Config, Security Hub ו-SNS
  • חשיבות ניטור ובקרה: זמינות, ביצועים, אבטחה, תאימות וניהול עלויות
  • תהליך ניטור: Data Collection, Analysis, Alerts, Reports/Dashboards ו-Automation
  • יתרונות שירותי ניטור AWS ואתגרי ענן: דינמיות, מורכבות וכמות נתונים גדולה
  • CloudTrail: מי ביצע קריאה, מתי, מאיפה, איזה שירות ומה הפעולה
  • Event History, מסך רשימת אירועים, דוגמת CreateSecurityGroup ופרטי אירוע
  • מבנה JSON של CloudTrail: userIdentity, eventTime, eventSource, eventName, sourceIPAddress ו-requestParameters
  • יצירת Trail, Multi-Region Trails, CLI/API, S3, CloudWatch Logs ו-Data Events
  • Inspector: ניהול פגיעויות, EC2, ECR, Lambda, CVE, CVSS והמלצות תיקון
  • GuardDuty: זיהוי איומים בזמן אמת ממקורות CloudTrail, VPC Flow Logs, DNS ועוד
  • השוואת Inspector מול GuardDuty: חולשות ידועות מול איומים והתנהגות חשודה
  • CloudWatch: Metrics, Logs, Alarms, Dashboards, Events, Lambda, SNS ו-SQS
  • AWS Config: מכונת זמן לתצורה, Rules, Compliance, Non-compliant ודוגמת פורט 22 פתוח לעולם
  • Security Hub: Single Pane of Glass, איסוף ממצאים, בדיקות תאימות, דוחות ואינטגרציות
  • SNS: Pub/Sub, Publisher, Topic, Subscribers, SQS, Lambda, Email, SMS ו-HTTP/S Webhooks
  • מסע התראה מסכם: אירוע, תיעוד, ריכוז, ניתוב והתראה
